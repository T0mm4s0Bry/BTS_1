1)Bootstrap è un framework front-end gratuito per uno sviluppo web più rapido e semplice

2)Per creare una pagina bootstrap devi importare 2 codici <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet>"
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js">

3)Il CDN è Una rete per la distribuzione dei contenuti.
Il CDN serve nel momento in cui un utente richiede un file,perché questo verrà servito dal server più vicino a loro, il che porta anche a tempi di caricamento più rapidi.

4)Questo file meta <content="width=device-width, initial-scale=1er> serve per garantire il rendering e lo zoom tattile corretti

5)La differenza tra conteiner e conteiner-fluid si presenta nel momento in cui si evidenzia una larghezza,
perché quando utilizziamo un container prendiamo una larghezza fissa, invece quando usiamo un container-fluid prendiamo la larghezza di tutta la pagina
